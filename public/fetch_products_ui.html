Age-Based Products Fetcher

Fetch and categorize 300-400 products from Bol.com API by age ranges

How it works: This tool fetches products from the Bol.com API, filters out age-related products (anniversaries, birthdays), and categorizes them into age ranges (18-25, 25-35, 35-50, 50-65, 65+). Results are saved to age_based_products.json.
Features
  • Fetches products from multiple search queries
  • Filters out birthday/anniversary products
  • Requires product images (skips products without images)
  • Categorizes by age range and price
  • Saves to JSON for fast loading
  • Real-time progress updates

Instructions
  1. Make sure your .env file has BOL_CLIENT_ID and BOL_CLIENT_SECRET set
  2. Click "Start Fetching Products" to begin the process
  3. The script will fetch products from various search queries
  4. Products are automatically categorized by age range and price
  5. Results are saved to public/age_based_products.json
  6. The dashboard will automatically use these products for personalized suggestions
  7. Delete this file after use for security: public/fetch_products_ui.html
